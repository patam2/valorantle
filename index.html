Valorantle explanation
Guess a Valorant player. And see where the clues lead you.

The player you enter will be inserted to the table, and all items will be color coded according to their accuracy compared to the correct answer.

Table header explanations:
Country - Country the player is from. Can be different from the country of current residence. If yellow, then the answer player plays in the same region as the guessed one.
Team - Where the player plays on. If green then you got the correct team.
Agents - Most commonly played agents. Sorted by the order of most played. If the agent has a yellow border, then the answer player plays the same agent but not at the same frequency. If the agent has a green border, then the answer player plays the same agent the same frequency.
IGL - If the player is an IGL or not. If green border, then the answer player has the same IGL status as guessed one.
